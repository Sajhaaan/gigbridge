# GigBridge

A mobile application that connects workers with businesses for gig work opportunities.

## Features

- User authentication (login/signup)
- Job posting and searching
- Real-time messaging between workers and businesses
- Profile management
- Job history and reviews

## Tech Stack

- React Native with Expo
- TypeScript
- React Navigation
- AsyncStorage for local storage
- Axios for API calls

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn
- Expo CLI
- iOS Simulator (for Mac) or Android Studio (for Android development)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/gigbridge.git
cd gigbridge
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```

4. Run on your preferred platform:
- Press `i` to open in iOS simulator
- Press `a` to open in Android emulator
- Scan the QR code with Expo Go app on your physical device
